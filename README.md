# Featured Project

## 군수과장 벡터양!

[**군수과장 벡터양!**](https://krissvector.moe/)은 모바일 게임 '소녀전선'의 군수 시스템을 서포트하는 웹 어플리케이션입니다. DFS를 활용한 전수탐색 알고리즘으로 사용자의 일정에 최적화된 작전 조합을 찾아줍니다. 전세계 5개국어로 서비스 중. [자세히보기](https://github.com/Phryxia/vector_logistics)

- JavaScript
- Linear Algebra and Cosine Similarity
- Netlify



## 소녀전선 인형 검색엔진

[**소녀전선 인형 검색엔진**](https://gfcombinator.netlify.app/)은 모바일 게임 '소녀전선'의 유닛을 검색할 수 있는 웹 어플리케이션입니다. 수작업으로 수집한 300여개가 넘는 유닛의 정보를 효과적으로 검색할 수 있습니다. [자세히보기](https://github.com/Phryxia/gf-combinator)

- TypeScript
- Netlify



## 혼공의 정석

[**혼공의 정석**](https://hongong.netlify.app/)은 사용자가 입력한 구조적 문서를 기반으로 사지선다 문제를 만들어주는 웹 어플리케이션입니다. 트리로 표현한 1차논리를 기반으로, 파싱과 탐색을 이용하여 문제를 만듭니다. [자세히보기](https://github.com/Phryxia/hongong)

- TypeScript
- Netlify

- LLParser and Tree Search
